An automation for keeping some 3rd party apps up-to-date on macOS. These scripts will require upkeep as vendors change their URLs. 
Currently supports:
  "Google Chrome",
	"Cyberduck",
	"Firefox",
	"Microsoft Edge",
	"Microsoft Office",
	"PowerShell",
	"Slack",
	"zoom"
  
Run install-powershell.sh first before running the automation script.
